# 🎧 Generador de Subtítulos de YouTube

Este es un proyecto de práctica donde se generan subtítulos a partir de videos de YouTube usando inteligencia artificial.

El sistema descarga el audio del video y lo transcribe utilizando Whisper.

## 🚀 Funcionalidades

* Generar subtítulos automáticamente
* Buscar palabras dentro de los subtítulos
* Descargar subtítulos en formato .txt

## 🛠️ Tecnologías

* Python
* Flask
* Whisper
* yt-dlp
* JavaScript

## 📌 Notas

Proyecto realizado con fines de aprendizaje y práctica.
